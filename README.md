### 项目说明：
使用 Koa2 + MongoDB+ Ejs 实现图书增删改查
### 开发环境：
Node.js + MongoDB + koa2 + Ejs

### 如何使用：
* 首先`sudo mongod`启动MongoDB服务
* `cnpm install` 进行依赖模块安装
* 安装完成后，`npm run dev` 启动应用，浏览器打开 `http://localhost:8000` 即可预览
